Classification Uncertainty
---

## 🧐 About

An easy-to-run implementation using PyTorch for the paper
"
[*Trusted Multi-View Classification*](https://arxiv.org/pdf/2102.02051.pdf)
".

## 🎈 Usage

**Requirements**

+ python 3.8.8
+ numpy 1.19.2
+ pytorch 1.7.1
+ scikit-learn 0.24.1

**Running**

```shell
python main.py
```
